# 🧠 Vivir Plenamente – Backend

Backend del proyecto **Vivir Plenamente**, una plataforma web orientada a adultos mayores, enfocada en brindar información clara, accesible y fácil de navegar.

🔗 Demo: https://www.vivirplenamente.com.ar

---

## 📌 Sobre el proyecto

Este backend fue desarrollado como parte del rediseño del sitio “Vivir Plenamente”, un programa radial de Corrientes.

El objetivo principal fue transformar un sitio desactualizado en una plataforma moderna, permitiendo gestionar contenido de forma dinámica y escalable.

---

## 🧩 Funcionalidades

- Gestión de artículos y noticias
- Panel de administración autogestionable
- Estructura escalable para crecimiento futuro
